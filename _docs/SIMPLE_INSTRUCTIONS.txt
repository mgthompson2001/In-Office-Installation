CCMD Bot Master - Simple Installation Instructions
==================================================

FOR IT ADMINISTRATORS:
======================

1. Copy this entire "In-Office Installation" folder to each user's computer
2. Put it on their Desktop or in Documents folder
3. Tell users to double-click "install.bat" to install everything
4. That's it! The installer does everything automatically.

FOR END USERS:
==============

1. Double-click "install.bat" in this folder
2. Wait for installation to complete (takes 2-5 minutes)
3. Look for "CCMD Bot Launcher" on your desktop (red pillar "I" icon)
4. Double-click it to start using the bots
5. Select a bot from the list and click "Launch Selected"
6. No password required for normal usage!
7. Your computer is automatically registered for updates!

WHAT YOU NEED:
==============

- Python 3.8+ installed (from python.org)
- Google Chrome browser installed
- Internet connection
- Login credentials for Penelope and TherapyNotes

WHAT THE INSTALLER DOES:
========================

- Installs all required Python packages
- Creates desktop shortcut for easy access
- Tests everything to make sure it works
- Shows you exactly what's happening

TROUBLESHOOTING:
================

If installation fails:
- Make sure Python is installed from python.org
- Make sure Chrome browser is installed
- Try running Command Prompt as Administrator
- Contact IT support with the error message

If bots don't work:
- Check that you have the correct login credentials
- Make sure internet connection is working
- Try with a small test file first
